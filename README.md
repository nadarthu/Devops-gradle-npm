# DevOps LN02 Gradle NPM

A collection of HTML demonstrations showcasing various JavaScript charting and animation libraries.

## Libraries Used

This project demonstrates the following JavaScript libraries:

- **Anime.js** - A lightweight JavaScript animation library
- **ApexCharts** - Modern & Interactive Charts
- **Chart.js** - Simple yet flexible JavaScript charting library
- **Plotly.js** - Open-source JavaScript charting library

## Files

- `anime.html` - Demonstrates Anime.js animations
- `chart.html` - Shows Chart.js charting capabilities
- `line.html` - Line chart example
- `plot.html` - Plotly.js interactive plots

## Getting Started

1. Clone or download the repository
2. Open any of the HTML files in your web browser to view the demonstrations

## Dependencies

The project uses the following npm packages:

- `animejs`: ^3.2.2
- `apexcharts`: ^5.10.3
- `chart.js`: ^4.5.1
- `plotly.js-dist`: ^3.4.0
